# 从上往下打印二叉树 #
## 题目描述 ##
从上往下打印出二叉树的每个节点，同层节点从左至右打印。

### 遍历两次解法 ###
- **解题思路**

	从根节点开始，先左子树的根，再右子树的根分别入队，再依次出队
	
- **时间复杂度**

	**O(n)** 令链表长度为n
	
- **空间复杂度**

	**O(n)**， 使用的额外空间大小与n有关，队列最大时占用O(n/2)
